 Electro_Shock Pro Services

This repository contains the backend services for Electro_Shock Pro, handling API requests and data management using Dexie.js for local storage.

 Project Overview
The backend is built with Node.js and Express, providing API endpoints for:
- User Authentication
- Calculation Management
- Data Export

 Design Documents
All design documents, including the database design, are in the `design/` folder.
